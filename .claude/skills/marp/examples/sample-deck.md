# Content Layout - Dark

- This is the default body slide layout
- H1 gets a subtle rule underline
- Body text is GT Standard M Light at 20px
- Bullets have comfortable spacing for readability

**Key points** are highlighted with medium weight.

> Blockquotes use the accent-primary blue left border

<!-- _class: "content light" -->

# Content Layout - Light

- Same layout, light mode palette
- Background shifts to warm off-white (#EEECEA)
- Text becomes dark (#1A1A1A)
- Rules and borders shift to light grey (#D8D4D0)

**Key points** remain visually distinct.

> Blockquotes adapt to the light palette
