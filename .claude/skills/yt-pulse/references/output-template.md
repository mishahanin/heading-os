## HTML Design System

Create `$PULSE_DIR/yt-pulse-brief.html` with styled layout.

Design system (consistent with ceo-intel):
- Font: Segoe UI, Helvetica Neue, Arial, sans-serif
- Body: 10pt, line-height 1.55
- Page: A4, 20mm margins, @media print ready
- Colors: #1a1a1a (text), #c77b30 (accent/highlights), #2980b9 (links), #f7f3ed (callout bg)
- Header: "31 Concept - YT Pulse Intelligence" + date + query + timeframe
- Hot Topics section: numbered list with bold topic names, video count in small pill/badge
- Top Video: card with light border, metadata strip (channel, views, duration, age), thumbnail if available
- Key Takeaways: bullet list inside the video card
- Content Summary: regular paragraphs
- Notable Quotes: styled blockquotes with left border
- Watch Recommendation: bold badge-style label
- Relevance: callout box with #f7f3ed background, 3px left border #c77b30
- Runners-Up: clean table with alternating row backgrounds (#f9f9f9)
- Footer: "Generated by /yt-pulse" + date
